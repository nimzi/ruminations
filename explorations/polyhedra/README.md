# Polyhedra Explorations

## Current Investigations

### Dihedral Angle of the Regular Dodecahedron
**File**: `dihedral.tex`

A comprehensive derivation of the dihedral angle of the regular dodecahedron using pentagon folding and Rodrigues' rotation formula. The investigation:

- Sets up three regular pentagons sharing a common vertex
- Uses analytical geometry and rotation matrices
- Exploits mirror symmetry to reduce the problem
- Derives the exact result: δ = π - arctan(2) ≈ 116.57°
- Provides both analytical and numerical verification

**Key concepts**: Regular polyhedra, dihedral angles, Rodrigues' rotation formula, golden ratio geometry, pentagon folding
