# Explorations

This directory contains individual mathematical investigations and their computational implementations.

Each exploration typically follows this structure:
- Problem statement and motivation
- Mathematical background and theory
- Computational approach and implementation
- Results, analysis, and visualizations
- Conclusions and future directions

## Organization

Explorations are organized by topic or mathematical area, with each investigation in its own subdirectory containing:
- `README.md` - Overview and problem statement
- `theory.md` - Mathematical background
- `implementation/` - Code and computational work
- `results/` - Output, data, and analysis
- `references.md` - Related papers and resources
